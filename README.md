# Portfolio

This test repository now includes a **web-enabled chatbot** (ChatGPT-style interface) with live internet search.

## Features
- Simple chat UI similar to tools like ChatGot.
- Server-side internet access via DuckDuckGo web search.
- Source links shown for each answer.
- No API key required.

## Run locally
```bash
npm start
```

Open: `http://localhost:3000`

## Example prompt
- `What are the latest updates in React?`
- `Compare Next.js and Remix for SEO`
